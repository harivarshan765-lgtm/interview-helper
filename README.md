# Internship Assistant — Hari Harivarshan

உங்கள் personal internship helper website. English communication, meetings, emails — எல்லாத்துக்கும் help!

## எப்படி Open பண்ணுவது?

1. `index.html` file-ஐ double-click பண்ணுங்கள்
2. Browser-ல் (Chrome/Edge) open ஆகும்
3. அவ்வளவுதான்! Internet optional (fonts மட்டும் online)

## Features

| Tab | Use Case |
|-----|----------|
| **Live Meeting** | Meeting-ல் question வந்தா instant English answer |
| **Translate** | English text → Tamil word/phrase explanation |
| **Email Writer** | Professional email templates |
| **Marketing Terms** | CTR, ROI, Meta Ads — Tamil explanation |
| **English Fix** | Common mistakes correct பண்ணும் |
| **Meeting Notes** | Notes paste → summary |
| **Quick Phrases** | Daily useful English phrases |

## Mobile-ல் Use பண்ண

Phone browser-லும் work ஆகும். ☰ menu button press பண்ணி tabs switch பண்ணுங்கள்.

## Tips

- **Live meeting:** "Live Meeting" tab → question type select → Copy answer → speak
- **Email:** Details fill பண்ணி → Generate → Copy → send
- **Notes:** Meeting முடிஞ்சதும் notes paste → Summarize
- **Complex translation:** Cursor chat-ல் full AI help available
